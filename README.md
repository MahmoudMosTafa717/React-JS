# React-JS Projects Journey

This repository contains a collection of React projects built as part of my learning journey. It goes from basic React principles all the way to advanced full-stack patterns.

## Projects List

- **[Pizza Menu](./03-pizza-menu)**: A simple React application to display a pizza menu. Focuses on core React concepts such as Components, Props, and JSX.
- **[Travel List](./05-travel-list)**: An application to help users pack for their travels. Focuses on `useState` and essential state management in React.
- **[Eat-n-split](./06-eat-n-split)**: A practice project for splitting bills among friends. Emphasizes component composition and lifting state up.
- **[usePopcorn](./07-usepopcorn)**: A movie search and rating app. Focuses on `useEffect`, data fetching, and creating custom hooks.
- **[Classy Weather](./09-classy-weather)**: A weather forecasting application built using older React Class Components, providing historical context on how React used to be written.
- **[Quiz App](./10-react-quiz)**: A React-based quiz application. Focuses heavily on the `useReducer` hook for complex state management.
- **[WorldWise](./11-worldwise)**: A global travel tracking application. Focuses on React Router, the Context API, and performance optimization (memoization).
- **[Fast React Pizza](./16-fast-react-pizza)**: A modern pizza ordering app. Focuses on React Router data loading, Redux Toolkit for global state, and styling with Tailwind CSS.
- **[The Wild Oasis](./17-the-wild-oasis)**: The largest application in the series: A full-stack cabin management app. Focuses on React Query, Styled Components, Supabase integration, Authentication, and advanced compound component patterns.
